Syarat & Ketentuan Pengumpulan

Mohon perhatikan ketentuan berikut sebelum mengunggah tugas Anda:

  • Format file yang diizinkan: PDF, DOC, DOCX, ZIP, RAR, atau 7Z.
  • Ukuran file maksimal yang dapat diunggah adalah 12 MB.
  • Penamaan file harus mengikuti format yang ditentukan:
    [Nama Lengkap]_[NIM]_[Mata Kuliah]_[Nomor Pertemuan]
    (Contoh: Putra_1955201011_PBO_P1)
  • Kesalahan dalam format nama file dapat mengakibatkan tugas tidak tercatat atau dinilai.
  • Anda memiliki opsi untuk menghapus file yang salah unggah melalui menu Riwayat Upload.
